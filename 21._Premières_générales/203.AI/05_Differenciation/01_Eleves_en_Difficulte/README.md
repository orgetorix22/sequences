# DOCUMENTS POUR ÉLÈVES EN DIFFICULTÉ

Ce dossier contient le matériel adapté pour les élèves ayant besoin de soutien supplémentaire.

---

## CONTENU

### 1. Glossaire pré-rempli
**Fichier:** `GLOSSAIRE_PRECOMPLET.md`
- Termes clés avec traductions françaises
- Définitions simplifiées
- Exemples en contexte
**Utilisation:** Distribuer avant chaque nouvelle séance

### 2. Fiches de compréhension guidée
**Fichiers:**
- `AI_S02_Types_of_AI_GUIDE.md` - Questions guidées pour l'article S02
- `AI_S03_Deepfake_GUIDE.md` - Compréhension guidée avec aide
- `AI_S05_CE_QCM.md` - Version QCM de l'évaluation S05

**Utilisation:** Alternative aux questions ouvertes pour élèves A1/A1+

### 3. Tâche finale adaptée
**Fichier:** `AI_S09_Tache_Finale_90mots.md`
- Consigne réduite à 90-100 mots (au lieu de 100-120)
- Plan fourni avec structure guidée
- Phrases-modèles pour démarrer
**Utilisation:** Pour élèves avec grandes difficultés d'expression écrite

### 4. Fiches d'aide grammaticale
**Fichier:** `AIDE_GRAMMAIRE.md`
- Rappel des modaux (should, must, can)
- Connecteurs logiques avec exemples
- Structures de phrases types
**Utilisation:** Support pendant les évaluations

## PRINCIPES DE DIFFÉRENCIATION

### Adaptations proposées:

1. **Vocabulaire:**
   - Glossaire pré-rempli distribué en amont
   - Définitions en marge des documents
   - Autorisation du dictionnaire bilingue

2. **Compréhension écrite:**
   - Questions guidées avec choix multiples
   - Numéros de lignes pour localiser les réponses
   - Questions reformulées plus simplement

3. **Expression écrite:**
   - Réduction du nombre de mots attendus
   - Plan fourni (intro, 2 arguments, conclusion)
   - Phrases-amorces pour démarrer chaque paragraphe

4. **Oral (débat):**
   - "Argument cards" préparées à l'avance
   - Possibilité de lire des notes
   - Questions directes pour encourager la participation

5. **Évaluations:**
   - Temps supplémentaire (+10 min)
   - QCM au lieu de questions ouvertes (si adaptations PAP/PPS)
   - Barème adapté (moins de pénalités sur la forme)

---

## COMMENT UTILISER CES DOCUMENTS

### Avant la séquence:
1. Identifier les élèves nécessitant ces adaptations (PAP, PPS, PPRE, ou élèves repérés en difficulté)
2. Préparer les documents adaptés et les imprimer
3. Informer les élèves des supports disponibles

### Pendant la séquence:
1. Distribuer le glossaire avant chaque nouvelle séance
2. Proposer les fiches guidées en parallèle des documents standards
3. Encourager l'utilisation du dictionnaire
4. Prévoir du tutorat par pairs si possible

### Pour les évaluations:
1. Distribuer la version adaptée (QCM, tâche réduite, etc.)
2. Accorder le temps supplémentaire prévu
3. Adapter le barème si nécessaire (selon PAP/PPS)
