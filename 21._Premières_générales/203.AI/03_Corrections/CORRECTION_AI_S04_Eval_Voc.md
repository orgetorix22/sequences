## Part B - Fill in the blanks /5 pts (0.5 pt each)

1. ChatGPT is an example of **narrow AI** (ou **artificial intelligence** accepté).
2. **Artificial intelligence** is essential to voice assistants like Siri and Alexa.
3. **General AI** would be able to think like a human.
4. Understanding AI **literacy** is important for the future.
5. When you ask ChatGPT a question, you give it a **prompt**.
6. Some schools decided to **ban** AI tools.
7. Students need **foundational skills** before using advanced technology.
8. As soon as something is banned, it becomes **forbidden fruit**.
9. Developing a system with its own **consciousness** is still far away.
10. It is **crucial** that we learn how to use AI responsibly.

**Variantes acceptables:**
- Question 1: "artificial intelligence" est acceptable (0.5 pt)
- Question 2: "narrow AI" peut être accepté si contexte correct (0.25 pt)

## Part C - Translation /5 pts (0.5 pt each)

1. available → **disponible**
2. smart → **intelligent / malin / astucieux**
3. currently → **actuellement / en ce moment**
4. skills → **compétences / capacités**
5. task → **tâche**
6. to solve → **résoudre**
7. to carry out → **effectuer / réaliser / accomplir / mener à bien**
8. chatbot → **chatbot / agent conversationnel** (les deux acceptés)
9. to design → **concevoir / créer**
10. crucial → **crucial / essentiel**

**Variantes acceptables:**
- Les synonymes proches sont acceptés
- Orthographe: -0.25 pt pour faute mineure, -0.5 pt pour faute majeure

**Total Part C: /5 points**
